# Electroneum DApp Monorepo (React + TS + Wagmi + Hardhat)

Welcome to your Electroneum DApp monorepo! This project is pre-configured with a React (Vite) TypeScript frontend and a Hardhat smart contract development environment.

## 📁 Project Structure

- **`frontend/`**: Web application (React + TS + Wagmi + Viem + Tailwind CSS v4)
- **`smart-contract/`**: Hardhat workspace for contract development

## 🚀 Workspace Commands

Run these commands from the root directory:

- **`npm run dev`**: Start the frontend development server
- **`npm run build`**: Build the frontend for production
- **`npm run start`**: Start the production server for the frontend
- **`npm run hardhat`**: Start a local Hardhat network node
- **`npm run compile`**: Compile your smart contracts
- **`npm run test`**: Run smart contract tests

## ⚙️ Prerequisites

Before you start, ensure you have:
1. **Node.js** (v18 or higher)
2. **Ankr API Key** (set in your environment variables for RPC access)
3. **Private Key** (for contract deployment, never share this!)
